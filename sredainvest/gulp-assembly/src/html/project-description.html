Описание объекта

Готовый объект недвижимости общей площадью 1204 м2. Здание из 2х этажей расположено вдоль Пашинской ул. С автомобильным трафиком ХХХ авто/час. В непосредственной близости автобусная остановка, пешеходный переход, густонаселенный район. В радиусе ХХХ м проживает ХХХ чел. Здание полностью построено. Ввод в эксплуатацию планируется в автусте 2020г.
